# Lazarus
Just a Game Launcher!
## What is lazarus?
Lazarus is game launcher like other well known such as Steam or even Epic Games!

Lazarus Doesn't collect any sort of data about you nor sell some of!

## How to use
### Running Without Building
Open the folder in a terminal

Run this command

```npm run start```

Enjoy!
### Building yourself
Open the folder in a terminal

Make sure you have python installed

run this command

```python3 build.py```

Follow the instructions

All done!
